# 01 SQL/RDBMS

## 복습 목적

- SQL 기본 문법 재정리
- Oracle, MySQL, PostgreSQL 사용 경험 정리
- DB 환경 설정과 사용 도구 기억 보존
- 개념을 내 말로 설명할 수 있도록 정리

## DB 학습 기준

- 초기 SQL 학습: Oracle
- JSP/MyBatis 실습: MySQL
- Spring Boot/JPA 실습: PostgreSQL
- 현재 복습 기준: PostgreSQL
- DB별 차이점은 필요한 시점에 별도 정리
