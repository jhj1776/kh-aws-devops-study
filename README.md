\# kh-aws-devops-study



복습용 개인 학습 저장소



\## 학습 목적



\- SQL, Java, Spring, React, AWS DevOps 흐름을 다시 정리

\- 각 단계별 실습 코드와 회고를 커밋

\- 클라우드 배포 가능한 풀스택 프로젝트 생성



\## 학습 루틴



\- 월요일: 새 개념 학습 및 실습

\- 금요일: 실습 마무리 및 회고 정리



\## 커리큘럼



| 단계 | 주제 |

|---|---|

| 01 | SQL/RDBMS |

| 02 | Java Basic |

| 03 | JDBC/DB Modeling |

| 04 | HTML/CSS/JavaScript |

| 05 | Servlet/JSP |

| 06 | Spring MVC/MyBatis |

| 07 | React SPA |

| 08 | AWS DevOps |

| 09 | Spring REST API/JPA/JWT/Redis |

| 10 | Final Project |



\## 커밋 규칙



| 타입 | 의미 |

|---|---|

| docs | 문서 작성 및 수정 |

| practice | 학습 실습 코드 작성 |

| feat | 기능 추가 |

| fix | 버그 수정 |

| refactor | 코드 구조 개선 |

| test | 테스트 코드 작성 |

| chore | 설정, 폴더 구조, 빌드 작업 |
